# Python_for_ESADE
Teaching materials for the ESADE Business Analytics course focused on Introduction to Python. Includes notebooks, exercises, and examples to help students build foundational programming and analytical skills.

## Contents

### Session 1: Introduction to Python
Introduction to core Python programming concepts including:
- Basic syntax and data types
- Variables and operators
- Control structures (if/else, loops)
- Functions and basic data structures
- Working with files and basic I/O

**Materials**: Lecture notebooks, exercises, and solutions

### Session 2: Data Wrangling
Learning data manipulation and analysis with pandas:
- Introduction to pandas DataFrames and Series
- Data cleaning and transformation
- Basic plotting and visualization
- Pivot tables and data aggregation
- Working with CSV files and datasets

**Materials**: Lecture notebooks, exercises with solutions, and sample datasets

### Session 3: Descriptive Statistics
Deep dive into data visualization and statistical analysis:
- Advanced plotting techniques
- Descriptive statistics fundamentals
- Measures of central tendency (mean, median, mode)
- Measures of dispersion (variance, standard deviation)
- Data distribution analysis

**Materials**: Lecture notebooks, exercises with solutions, and statistical datasets

### Session 4: Introduction to Causality
A/B testing and causal inference:
- Fundamentals of causal thinking
- A/B testing methodology
- Statistical significance and hypothesis testing
- Introduction to causal inference concepts
- Practical applications in business analytics

**Materials**: Lecture notebooks and exercises with solutions
